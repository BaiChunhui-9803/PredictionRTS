**与文档高度契合的学术论文整理**

文档的核心学术定位是**Offline Model-Based Planning on Experience Transition Graphs (ETG)**：从RL轨迹中统计聚合构建显式、表格式的经验转移图（离散化状态 + 动作边 + 多维统计属性），在图上执行**Beam Search规划**，结合**滚动推演（Receding-Horizon Planning with BTB Recovery）**，实现数据生产（RL采样）与决策规划的完全解耦，同时强调可解释性、可视化与状态抽象。

该框架在学术谱系中处于**离线 + 显式模型 + 图搜索规划**的较少见但合理的交叉位置。它区别于Offline RL（学习策略函数）、Online MBRL（在线交互）和纯Model-Free方法，强调“从经验数据直接构建转移模型 + 图上启发式搜索”。

以下是我根据文档技术路线（ETG构建、状态抽象、Beam Search规划、BTB偏差恢复、滚动推演、解耦范式）筛选并整理的最契合学术论文（优先2020年后高影响力工作，按相关度排序）。我将每篇论文的**核心贡献**、**与文档的契合点**、**差异/互补**简要说明，并附arXiv/链接（便于进一步阅读）。这些论文共同构成了文档框架的学术基础，但文档的**显式ETG + Beam Search + BTB**组合具有明显创新性（多数论文仍依赖神经网络世界模型或策略优化，而非纯经验图搜索）。

### 1. **高度契合：图结构世界模型 + 图搜索规划**
这些论文直接使用“图”作为世界模型，并通过图搜索实现规划，与ETG + Beam Search高度一致。

- **World Model as a Graph: Learning Latent Landmarks for Planning (L3P)**  
  Lunjun Zhang, Ge Yang, Bradly C. Stadie (ICML 2021, arXiv:2011.12491)  
  **核心**：提出图结构世界模型（节点=latent landmarks，边=多步可达性估计，从Q函数提炼），在图上执行图搜索规划。  
  **契合点**：与ETG完全一致——将RL经验抽象为稀疏图，利用图搜索实现长时序规划；强调“model-free RL的鲁棒性 + 图搜索的泛化能力”。支持高维连续控制任务。  
  **差异/互补**：使用latent空间聚类（而非文档的BK-Tree离散化）；边是Q函数蒸馏的reachability，而非文档的显式统计计数（visits、win_rate、quality_score）。文档的BTB恢复可作为其滚动推演的扩展。**最推荐参考论文**。

- **Value Memory Graph: A Graph-Structured World Model for Offline Reinforcement Learning**  
  Deyao Zhu et al. (arXiv:2206.04384, 2022)  
  **核心**：构建有向图MDP（顶点=图状态，边=图动作），从offline数据集直接构建并训练，用于规划/控制。  
  **契合点**：与ETG结构几乎一致（图=显式世界模型，离线构建）；动作翻译器对应文档的“高阶Markov查询”。  
  **差异**：仍偏向值函数/策略优化，而非纯Beam Search规划。可作为文档ETG的扩展（添加value memory）。

### 2. **高度契合：Offline Model-Based Planning 框架**
这些是文档“Offline Model-Based Planning”定位的直接奠基工作，强调“离线建模 + 直接规划（而非学策略）”。

- **Model-Based Offline Planning (MBOP)**  
  Arthur Argenson, Gabriel Dulac-Arnold (ICLR 2021, arXiv:2008.05556)  
  **核心**：从offline数据集学习世界模型，直接通过规划（MPC-style guided shooting）控制系统，无需在线交互或策略学习。  
  **契合点**：文档的“离线固定模型 + 规划决策”定位完全匹配；支持reward/constraint灵活修改（与文档滚动推演的灵活性一致）。实验在机器人任务上验证。  
  **差异**：使用NN动态模型 + 轨迹优化器（CEM/MPPI），而非文档的显式ETG + Beam Search。MOPP是其直接扩展。

- **Model-Based Offline Planning with Trajectory Pruning (MOPP)**  
  Xianyuan Zhan et al. (IJCAI 2022 / arXiv:2105.07351)  
  **核心**：基于MBOP，引入行为策略引导轨迹rollout + 不确定性剪枝（prune OOD轨迹），实现轻量级offline规划。  
  **契合点**：直接解决“offline限制 vs 高性能规划”的权衡；轨迹剪枝可视为文档BTB偏差恢复的类似机制（避免偏差路径）。  
  **差异**：仍依赖NN模型rollout，而文档是纯图搜索 + 预计算switching_map。**极强互补性**——文档的ETG可替换其动态模型。

### 3. **高度契合：表格/经验模型 + 状态抽象的Offline MBRL**
这些提供ETG的“经验统计聚合 + 离散化”理论基础。

- **Settling the Sample Complexity of Model-Based Offline Reinforcement Learning**  
  Gen Li et al. (2022, NeurIPS等会议版本；PDF: yuxinchen2020.github.io/publications/offline_modelbased.pdf)  
  **核心**：证明在tabular MDP上，从offline数据集构建empirical transition matrix（基于计数N(s,a,s')）并做pessimistic value iteration（VI-LCB），实现minimax-optimal sample complexity。  
  **契合点**：ETG的`transitions: s → a → s' → count`、`visits`、`avg_reward`等统计属性完全对应empirical MDP；文档的“经验频率近似MDP转移/奖励函数”在此有严格理论支撑。  
  **差异**：使用value iteration规划，而非Beam Search；适用于tabular/离散化场景，与文档状态抽象完美匹配。

- **Model-Invariant State Abstractions for Model-Based Reinforcement Learning** (及相关综述)  
  M. Tomar et al. (arXiv:2102.09850, 2021)；另见RAN Starre et al. “Model-Based Reinforcement Learning with State Abstraction: A Survey”（2022）  
  **核心**：系统调研状态抽象（aggregation、latent representation等）在MBRL中的作用，提出model-invariant抽象以提升泛化。  
  **契合点**：文档“连续状态 → 离散化（BK-Tree） + 抽象状态ID”正是state abstraction的核心；可直接指导ETG节点设计。  
  **差异**：侧重理论/泛化，文档将其工程化为ETG可视化与查询。

### 4. **次要但相关的：Beam Search / 滚动规划扩展**
- **Learning Linear Ranking Functions for Beam Search with Application to Planning** (Yuehua Xu et al., JMLR 2009)：学习排名函数指导Beam Search，用于规划问题。 与文档Beam Search + 多维评分（quality/future reward/win rate）高度一致，可作为评分函数学习的理论参考。
- 滚动推演与BTB：文档的Plan-Execute-Diverge + Branch-to-Backup借鉴MPC文献（receding-horizon control）和CPU分支预测；MBOP/MOPP中的轨迹优化已有类似replan机制，可作为工程扩展。

### 总结与建议
- **最强契合集群**：L3P（图世界模型+规划） + MBOP/MOPP（offline planning框架） + 表格empirical MDP论文（ETG统计基础） + 状态抽象综述，共同覆盖文档90%以上的技术路线。
- **文档创新点突出**：现有工作多依赖NN世界模型或策略学习，而文档的**纯经验ETG + 确定性Beam Search + BTB + 四维路径评估 + 可视化系统**实现了更强的可解释性、计算可预测性和解耦性，在real-world工业控制/游戏等场景中具有独特优势。
- **进一步阅读建议**：优先阅读L3P和MBOP两篇（均有开源代码）；若需理论基础，可结合Gen Li et al.的sample complexity分析验证ETG的有效性。
