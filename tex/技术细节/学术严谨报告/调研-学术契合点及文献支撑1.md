**与文档高度契合的学术论文整理**

该文档的核心框架是**离线模型规划（Offline Model-Based Planning）**，其独特之处在于：
- 从 RL 轨迹统计聚合构建**经验转移图（Experience Transition Graph, ETG）**（显式、非神经网络的表格/图结构世界模型，包含 visits、avg_step_reward、win_rate、quality_score 等统计属性）；
- 采用**状态抽象/离散化**；
- 在图上执行**束搜索（Beam Search）规划** + **滚动推演（Receding-Horizon Planning with BTB 偏差恢复）**；
- 强调数据采集（RL 交互）与决策规划（图搜索）的完全解耦、可解释路径、确定性规划。

这占据了“离线 + 显式模型 + 基于搜索的规划”这一较少见的学术位置（区别于 Offline RL 的策略学习和 Online MBRL）。以下是与该框架**高度契合**的代表性学术论文，我按相关度排序整理，并说明契合点（基于公开 arXiv/OpenReview 等信息）。这些论文主要来自 ICLR、NeurIPS、IJCAI 等顶级会议，聚焦图结构世界模型、离线模型规划、状态抽象转移图等。

### 1. **高度核心契合：图结构世界模型 + 离线规划**
- **Value Memory Graph: A Graph-Structured World Model for Offline Reinforcement Learning** (Zhu et al., ICLR 2023, arXiv:2206.04384)  
  **最契合的论文**。提出 VMG（Value Memory Graph），从离线数据集构建**有向图 MDP**（节点=抽象图状态，边=图动作），状态空间有限且离散化，直接在图上运行价值迭代（value iteration）进行规划。完全匹配文档的 ETG 定义（经验驱动的转移图）、状态抽象、离线固定模型 + 图上规划。强调简单、离散、可规划的世界模型，与文档“最简形式的显式 World Model”高度一致。

- **Model-Based Offline Planning (MBOP)** (Argenson & Dulac-Arnold, arXiv:2008.05556, 2020/2021)  
  直接提出“Model-Based Offline Planning”范式：从离线日志学习模型，然后**直接通过规划（而非策略优化）**控制系统。后续扩展包括 **Model-Based Offline Planning with Trajectory Pruning (MOPP)** (Zhan et al., IJCAI 2022 / NeurIPS Offline RL Workshop 2021)，使用轨迹剪枝在离线模型上规划，与文档的 Beam Search + 路径评估（置信度、质量、胜率、奖励）高度类似。文档的“Offline Model-Based Planning”学术定位几乎与此完全重合。

### 2. **状态抽象 + 抽象转移图（Abstract State Transition Graphs）**
- **Abstract State Transition Graphs for Model-Based Reinforcement Learning** (de Mendonça et al., 2018, IEEE)  
  提出 ASTG（Abstract State Transition Graphs），通过融合结构相似状态创建**抽象转移图**，用于模型-based RL。直接对应文档的“状态抽象与离散化 → 经验转移图构建”流水线，以及 ETG 的节点（抽象状态 ID）+ 边（动作+转移频次）设计。

- **Highway Graph to Accelerate Reinforcement Learning** (Yin et al., arXiv:2405.11727, 2024)  
  在经验状态转移图上构建“Highway”（非分支转移序列），压缩模型以加速价值传播和规划。匹配文档的 ETG 统计聚合 + 图上高效搜索，强调从经验轨迹构建的转移图用于 model-based off-policy RL。

### 3. **离线 MBRL 中的规划与滚动推演相关工作**
- **Trajectory Transformer** (Janner et al., NeurIPS 2021)  
  将 RL 视为序列建模，在离线数据上用 Transformer + **Beam Search** 进行轨迹规划（而非策略学习）。与文档的“束搜索规划 + 路径树图可视化”高度一致（虽非纯图结构，但规划机制相同）。

- **Drifting MPC: Receding-Horizon Planning under Unknown Dynamics** (近期 arXiv 工作，2025/2026)  
  结合离线轨迹 + 生成模型的**滚动视界规划（Receding-Horizon）**，处理动态未知情况。与文档的 Plan-Execute-Diverge + BTB 偏差恢复机制（Branch-to-Backup）在理念上高度相关（虽 BTB 的 CPU 分支预测灵感较独特）。

### 4. **状态抽象与经验图的其他支撑工作**
- **Effective Reinforcement Learning through State Abstraction** (David Abel 等多篇工作，2018–2021) 及 **Model-Based Reinforcement Learning with State Abstraction: A Survey** (Starre et al., 2022)  
  系统性讨论状态抽象在 MBRL 中的作用（包括离散化、聚合），为文档的 BK-Tree 层次聚类 + 状态 ID 编码提供理论基础。

- **Graph-based State Representation** 系列（如 Waradpande et al., 2020；Xue et al., 2021）  
  利用 MDP 的转移图拓扑结构学习低维状态表示，与 ETG 的“从轨迹诱导的图结构”一致。

### 总结与创新点对比
- **最契合的学术定位**：VMG + MBOP/MOPP 几乎覆盖了文档的核心（离线 + 显式图模型 + 图上规划），证明该框架处于一个“完全合理但不常见”的位置（如文档对比表所示）。
- **文档的独特创新**（未在上述论文中完全覆盖）：
  - ETG 的**多维统计属性**（quality_score、win_rate 等综合效用，而非仅转移概率）；
  - **BTB 偏差恢复**（借鉴 CPU 分支预测的 switching_map + 精确+距离回退）；
  - **四维路径评估 + 可视化系统**（路径树、片段匹配、推演回放）；
  - **完全解耦 + 确定性规划**（规划结果可重复、可追溯）。
- Kandula (2023) 的 DKG（如文档 7.3 节提及）仅用于临床指南 QA，与 RL/规划无关，可忽略。
